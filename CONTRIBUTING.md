# Contributing

## Branching
- `main` — stable
- Feature branches: `feat/<issue>-<short-name>`
- Bugfix branches: `fix/<issue>-<short-name>`
- Docs: `docs/<issue>-<short-name>`
- Chores: `chore/<issue>-<short-name>`

Examples:
- `feat/12-neows-client`
- `fix/31-cache-key`
- `docs/6-readme-roadmap`

## Commit messages
Use Conventional Commits:
- `feat: ...`
- `fix: ...`
- `docs: ...`
- `chore: ...`
- `refactor: ...`

Optionally include issue reference:
- `feat: add NeoWs client (#12)`

## Pull Requests
- Keep PRs small and focused.
- Link issues using `Closes #<id>` in the PR description.
- Add screenshots for UI changes.

## Versioning
This project uses SemVer: `MAJOR.MINOR.PATCH`
- `0.x` — development
- `1.0.0` — first stable release
